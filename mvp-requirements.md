# Image Background Remover - MVP 需求文档

## 1. 项目概述

**产品名称：** Background Remover（待定）
**核心功能：** 用户上传图片，自动去除背景，下载透明背景PNG
**目标用户：** 电商卖家、内容创作者、普通用户
**技术栈：** Cloudflare Workers + Pages + Remove.bg API

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽/点击上传，格式 JPG/PNG/WebP，最大 10MB | P0 |
| 背景去除 | 调用 Remove.bg API 处理图片 | P0 |
| 结果预览 | 实时显示原图vs处理后对比 | P0 |
| 图片下载 | 一键下载透明背景PNG | P0 |
| 进度提示 | 处理中显示加载动画/进度 | P0 |

### 2.2 辅助功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 历史记录 | 本地 LocalStorage 保留最近10张 | P1 |
| 批量处理 | 一次上传多张图片队列处理 | P2 |
| 格式选择 | 下载时可选择 PNG/JPG/WebP | P2 |

---

## 3. 非功能需求

- **性能：** 处理时间 < 5秒（取决于图片大小和网络）
- **可用性：** Cloudflare Workers 全球边缘部署
- **成本：** Workers 免费额度 + Remove.bg 按调用计费
- **隐私：** 图片不存储，内存处理后即销毁

---

## 4. 技术架构

```
用户上传图片
    ↓
Cloudflare Worker 接收
    ↓
转发至 Remove.bg API（Base64）
    ↓
获取结果（透明PNG）
    ↓
返回给用户浏览器
    ↓
图片不落地，直接展示/下载
```
